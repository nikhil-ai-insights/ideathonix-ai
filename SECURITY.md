# 🔒 Security Policy

## Supported Versions

The following versions of Ideathonix AI are currently supported with security updates.

| Version | Supported |
| -------- | --------- |
| Latest   | ✅ Yes |
| Older Versions | ❌ No |

---

## Reporting a Vulnerability

The security of **Ideathonix AI** is important to us.

If you discover a security vulnerability, please report it responsibly instead of disclosing it publicly.

Please include the following details:

- Description of the vulnerability
- Steps to reproduce the issue
- Potential impact
- Screenshots or proof of concept (if applicable)
- Suggested mitigation (optional)

We will:

- Acknowledge your report as soon as possible.
- Investigate the issue.
- Work on a fix if the vulnerability is confirmed.
- Credit responsible disclosures where appropriate.

Please avoid publicly sharing security issues until they have been reviewed and resolved.

Thank you for helping keep Ideathonix AI secure. 🛡️

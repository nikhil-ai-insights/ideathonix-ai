# 🤝 Contributing to Ideathonix AI

First of all, thank you for your interest in contributing to **Ideathonix AI**! 🎉

We welcome contributions from developers, designers, AI enthusiasts, students, and innovators who want to help build a better platform for hackathon participants, startup founders, and creators.

---

## 🌟 Ways You Can Contribute

You can contribute by:

- 🐞 Reporting bugs
- 💡 Suggesting new features
- 🎨 Improving the UI/UX
- ⚡ Optimizing performance
- 🤖 Enhancing AI prompts and workflows
- 📖 Improving documentation
- 🧪 Writing tests
- 🔧 Fixing bugs
- 🌍 Improving accessibility and responsiveness

---

## 🚀 Getting Started

### 1. Fork the Repository

Click the **Fork** button at the top-right of this repository.

### 2. Clone Your Fork

```bash
git clone https://github.com/your-username/ideathonix-ai.git
```

### 3. Navigate to the Project

```bash
cd ideathonix-ai
```

### 4. Install Dependencies

```bash
npm install
```

### 5. Start the Development Server

```bash
npm run dev
```

---

## 🌿 Create a New Branch

Always create a new branch before making changes.

```bash
git checkout -b feature/your-feature-name
```

Example:

```bash
git checkout -b feature/improve-ui
```

---

## ✅ Commit Your Changes

Use clear and meaningful commit messages.

Examples:

```text
feat: add startup idea generator

fix: resolve responsive layout issue

docs: update README

refactor: optimize component structure
```

---

## 📤 Submit a Pull Request

1. Push your branch.
2. Open a Pull Request.
3. Clearly explain your changes.
4. Wait for review and feedback.

---

## 📝 Contribution Guidelines

Please ensure that:

- Your code is clean and readable.
- Follow existing project structure.
- Keep components reusable.
- Test your changes before submitting.
- Update documentation if required.
- Write meaningful commit messages.

---

## 🐛 Reporting Bugs

When reporting a bug, please include:

- Steps to reproduce
- Expected behavior
- Actual behavior
- Browser/device information
- Screenshots (if applicable)

---

## 💡 Feature Requests

When suggesting a feature, please explain:

- The problem you're solving
- Why the feature is useful
- Possible implementation ideas

---

## 📜 Code of Conduct

Please be respectful and professional.

We are committed to creating a welcoming, inclusive, and collaborative environment for everyone.

---

Every contribution—big or small—helps improve Ideathonix AI.

Thank you for helping us build a platform that empowers innovators around the world.
